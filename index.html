Dyck Paths Visualizer

See Source on GitHub

A dyck path is a sequence of steps from (0, 0) to (n, n) that lies strictly below, or touching, the diagonal line.

DISCLAIMER:
The greater the value of "n", the longer it will take to load the visualizer!
The web app may crash if "n" is too large!

(n has to be an integer > 0)
